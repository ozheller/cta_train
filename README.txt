This modules creates a block that will pull in the JSON feed from the CTA. You will need a API Key to access the feed.

CTA Documentation can be found at http://www.transitchicago.com/developers/traintracker.aspx
A CTA API Key can be obtained at http://www.transitchicago.com/developers/traintrackerapply.aspx 
Examples and uses can be found at http://www.transitchicago.com/developers/ttdocs/default.aspx
